## Descripción del proyecto

Este proyecto corresponde al desarrollo de un portafolio web personal, creado con el objetivo de presentar mi perfil profesional, habilidades técnicas, formación académica y proyectos desarrollados durante mi proceso formativo.

El portafolio fue diseñado bajo principios de diseño moderno, responsivo y accesible, permitiendo una navegación clara y una experiencia de usuario fluida tanto en dispositivos móviles como en escritorio.

El sitio busca reflejar no solo mis conocimientos técnicos, sino también mi compromiso con el aprendizaje constante y la mejora continua como desarrollador junior.

## Tecnologías utilizadas 

HTML5 – Estructura semántica del sitio
